    positive-value-radius

    Since 1.0.0
    @function positive-value-radius($radius) { ... }

    Description

    Função que substitui valores negativos para 0

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $radius

    Um conjunto ou um valor unitário númerico

    		Number— none

    Returns

    Number —

    Se houver um valor negativo retorna 0, senão o próprio valor.

    Example

    Função positive-value-radius

    positive-value-radius(-12px)
// 0px

positive-value-radius(42px)
// 42px

    Throws

    • Error related message

    border-radius

    Since 1.0.0
    @mixin border-radius($radius: $border-radius) { ... }

    Description

    Mixin para inserir border-radius em todos os cantos caso a variável $enable-rounded esteja definada como true

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $radius

    Um conjunto ou um valor unitário númerico

    		Number$border-radius

    Example

    border-radius

    @include border-radius(66px);
// border-radius: 66px;

@include border-radius(42px 66px);
// border-radius: 42px 66px;

    Output

    A saída vai ser um border-radius com o $radius

    Requires

    • [variable] enable-rounded
    • [function] positive-value-radius
